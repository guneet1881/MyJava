How it Works

This page is being served directly from your hard drive by a custom Java application using ServerSocket and Socket. It parses the raw HTTP HTTP GET requests and responds with the requested files along with standard HTTP response headers.

GET /index.html HTTP/1.1
Host: localhost:8080
Accept: text/html

Features

  • No external frameworks or libraries.
  • Multithreaded request handling using ExecutorService.
  • Basic MIME type detection for HTML, CSS, JS, Images.
  • Directory traversal protection (security feature).

Built with ❤️ using pure Core Java.
